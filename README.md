# ToDoList
## Лаба спп-2
### Тютюрин Егор

Приложение для создания списка задач. 

Реализованы функции:

* регистрации нового пользователя;
* авторизации существующего пользователя;
* добавления задачи в todolist
* редактирования задачи в todolist
* удаления задачи из todolist
